## 🚀 How to Run the Project

To run it on your local machine:

1. **Install Flutter:** Make sure you have the [Flutter SDK](https://docs.flutter.dev/get-started/install) installed.
2. **Get Dependencies:** Open your terminal in the project directory and run:
   ```bash
   flutter pub get
   ```
3. **Run the App:** You can run the app on an emulator, connected device, or Chrome by typing:
   ```bash
   flutter run
   ```
